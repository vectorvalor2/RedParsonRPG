# REDPARSON Safe Sim

**Confidence-gated 3D game engine shell** built with Three.js, featuring safety layers, semantic interlinking, undo/redo, and `.red` scene serialization.

---

## Installation

### Requirements

- Node.js ≥ 18
- npm or yarn

### Clone & Install

```bash
# 1. Unzip redparson-safe-sim.zip (or clone the repo)
cd redparson-safe-sim

# 2. Install dependencies
npm install

# 3. Command,Description
npm run dev,Start local development server
npm run build,Build production-ready files
npm run preview,Preview the production build
